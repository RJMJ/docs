# Requirements

**Database**
MySQL 5.5.3/MariaDB 5.5.20/Percona Server 5.5.8 or higher with PDO and an InnoDB-compatible primary storage engine.

**PHP**
PHP 5.5.9 or higher (PHP 7+ recommended)
